# datapackage_pipelines_estadisticas_judiciales

Módulo de [Data Package Pipelines](https://github.com/frictionlessdata/datapackage-pipelines) para automatización de datos de justicia.


## Install

```
# Ir a la carpeta de datapackages-pipelines

git clone https://github.com/datos-justicia-argentina/datapackage-pipelines-estadisticasjudiciales
python setup develop
```


## Usage

`dpp run  ./datapackage-pipelines-estadisticasjudiciales/datapackage_pipelines_estadisticasjudiciales/estadisticasjudiciales`
